Execution Steps
Step 1 : Install Node.js

Download and install:

Node.js

Step 2 : Check Installation

Open CMD and run:

node -v
npm -v

Step 3 : Install Angular CLI
npm install -g @angular/cli

Check Angular:

ng version

Step 4 : Create Angular Project
ng new login-app

Choose:

Routing : No
Style : CSS

Step 5 : Open Project Folder
cd login-app

Step 6 : Open VS Code
code .


Step 7 : Open File
src/app/app.html


Delete all code and paste HTML code.



Step 8 : Open File
src/app/app.ts

Delete all code and paste TypeScript code.


Step 9 : Save All Files
Ctrl + S


Step 10 : Run Project

Open terminal and run:

ng serve


Step 11 : Open Browser
http://localhost:4200
